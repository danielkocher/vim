# My personal vim configuration

## Installation

```
cd ~ && git clone --recursive https://github.com/danielkocher/vim.git
ln -s vim/vimrc .vimrc
ln -s vim .vim
```

## Update

```
git submodule update --init --recursive
```

## Status

Constantly improving.
